# manager-server
结合manager-fe,manager-fe是本项目的前端

```
1.没有koa2，有直接进入第2步
    全局安装脚手架
        yarn global add koa-generator
        或
        npm install -g koa-generator
2.安装依赖
    npm install
    或
    yarn
3.启动项目
    yarn start；yarn dev 会热启动
    或
    npm start manager-server
4.http://127.0.0.1:3001
```
